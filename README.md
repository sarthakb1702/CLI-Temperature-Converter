# Temperature Converter (Celsius ↔ Fahrenheit)

This is a simple Python CLI tool that converts temperatures between Celsius and Fahrenheit using command-line options.

## 🚀 Features

- Convert Celsius to Fahrenheit
- Convert Fahrenheit to Celsius
- Uses `argparse` for clean command-line interface
- Provides helpful usage messages

---

## 📝 Usage

### Run the converter:

```bash
python converter.py --c2f <Celsius value>
```
## OR
```bash
python converter.py --f2c <Fahrenheit value>
```
